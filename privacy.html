プライバシーポリシー

最終更新日: 2026年3月23日

andAugment（以下「当方」）は、当方が提供するソフトウェア製品（以下「本製品」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本製品がどのようなデータを取得・使用・保存するかを説明するものです。

本ポリシーは以下の製品に適用されます：

  • Video Auto Uploader for YouTube（VAU）
  • Keep Awake Pro（KAP）
  • よむよむ（Yomuyomu）
  • その他、当方が提供するすべての製品

1. 収集する情報

1.1 Webサイト（andaugment.com）

  • お問い合わせフォームを通じたお名前・メールアドレス・お問い合わせ内容

1.2 Video Auto Uploader for YouTube

  • Googleアカウント情報：YouTube認証（OAuth 2.0）を通じて、メールアドレスおよびYouTubeチャンネルへのアクセス権を取得
  • YouTubeアップロード権限：動画ファイルをYouTubeにアップロードするために使用
  • Google Sheets アクセス権限（任意）：アップロード履歴を記録するため、指定されたスプレッドシートへの書き込みアクセスを取得
  • ローカルファイル情報：監視フォルダ内のファイル名・ファイルパス

1.3 Keep Awake Pro

  • 個人情報の収集は行いません
  • 設定情報はローカルPCにのみ保存されます

1.4 よむよむ

  • 「小説家になろう」のAPI経由で小説データを取得（公開データのみ）
  • ブックマーク・閲覧履歴・読書設定はすべてデバイス内にのみ保存
  • 個人情報の収集・外部送信は行いません

2. 情報の使用目的

  • 製品の機能提供（YouTubeアップロード、スプレッドシート記録等）
  • 認証状態の維持およびトークンの自動更新
  • お問い合わせへの対応
  • 製品・サービスの改善

3. データの保存

  • 認証トークン（VAU）：OAuth 2.0のトークンは、ローカルPC上にWindows DPAPI（safeStorage）で暗号化して保存
  • 設定情報：すべての製品の設定はローカルPC/デバイス上にのみ保存
  • アップロード履歴（VAU）：ローカルPC上にのみ保存
  • 当方のサーバーにユーザーデータを送信・保存することはありません

4. 第三者への情報提供

当方は、ユーザーの個人情報を第三者に販売、貸与、または共有することはありません。製品は以下のサービスと通信しますが、これはアプリの機能を実現するためのみです：

  • YouTube Data API v3：動画のアップロード（VAU）
  • Google Sheets API：アップロード履歴の記録（VAU、任意機能）
  • Google OAuth 2.0：認証およびトークン管理（VAU）
  • なろうAPI：小説データの取得（よむよむ）

5. Google API Services User Data Policy への準拠

Video Auto Uploader for YouTube は Google API Services User Data Policy（Limited Use要件を含む）に準拠しています。

  • Google APIから取得したデータは、本ポリシーに記載された目的のみに使用します
  • Google APIから取得したデータを第三者に転送することはありません（ユーザーの明示的な同意、セキュリティ上の理由、または法的要件がある場合を除く）
  • Google APIから取得したデータを広告配信に使用することはありません
  • 人間がGoogle APIデータを閲覧するのは、ユーザーの同意がある場合、セキュリティ調査、法的義務への対応、またはデータが集計・匿名化されている場合に限ります

6. データの削除・アクセス権の取り消し

  • アプリ内ログアウト（VAU）：ログアウトボタンで保存されたトークンが削除されます
  • Googleアカウント設定：Googleアカウントのセキュリティ設定からアクセスを取り消せます
  • アプリのアンインストール：ローカルに保存されたすべてのデータが削除されます

7. セキュリティ

  • OAuth 2.0 PKCE（Proof Key for Code Exchange）による安全な認証
  • Windows DPAPI（safeStorage）によるトークンの暗号化保存
  • HTTPS通信のみ使用
  • Content Security Policy（CSP）による不正スクリプト実行の防止

8. Cookie

当サイトではCookieを使用していません。将来的にアクセス解析等の目的で使用する場合は、本ポリシーを更新し通知いたします。

9. 子どものプライバシー

本製品は13歳未満の子どもを対象としていません。13歳未満の方は本製品を使用しないでください。

10. ポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合は、アプリ内またはウェブサイトで通知します。
