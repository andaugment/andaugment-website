Privacy Policy

Last Updated: March 23, 2026

andAugment ("we", "us") respects the privacy of users of our software products ("Products") and is committed to protecting personal information. This Privacy Policy explains what data the Products collect, use, and store.

This policy applies to:

  • Video Auto Uploader for YouTube (VAU)
  • Keep Awake Pro (KAP)
  • Yomuyomu
  • All other products provided by us

1. Information We Collect

1.1 Website (andaugment.com)

  • Name, email address, and inquiry content through the contact form

1.2 Video Auto Uploader for YouTube

  • Google Account Information: Through YouTube OAuth 2.0, we access your email and YouTube channel
  • YouTube Upload Permission: Used to upload video files to YouTube
  • Google Sheets Access (optional): Write access to your specified spreadsheet for recording upload history
  • Local File Information: File names and paths within your watch folder

1.3 Keep Awake Pro

  • No personal information is collected. Settings are stored locally only.

1.4 Yomuyomu

  • Novel data retrieved via Narou API (public data only)
  • Bookmarks, reading history, and settings stored on-device only
  • No personal information collected or transmitted

2. How We Use Information

  • Providing product functionality (YouTube uploads, spreadsheet recording)
  • Maintaining authentication and automatic token refresh
  • Responding to inquiries
  • Improving products and services

3. Data Storage

  • Auth Tokens (VAU): Stored locally, encrypted with Windows DPAPI (safeStorage)
  • Settings: All product settings stored locally only
  • Upload History (VAU): Stored locally only
  • We do not transmit or store user data on our servers

4. Information Sharing

We do not sell, rent, or share personal information with third parties. Products communicate with these services solely to provide functionality:

  • YouTube Data API v3: Video uploads (VAU)
  • Google Sheets API: Upload history recording (VAU, optional)
  • Google OAuth 2.0: Authentication (VAU)
  • Narou API: Novel data retrieval (Yomuyomu)

5. Google API Services User Data Policy Compliance

Video Auto Uploader for YouTube complies with the Google API Services User Data Policy, including Limited Use requirements.

  • Data from Google APIs is used only for purposes described in this policy
  • We do not transfer Google API data to third parties (except with explicit consent, for security, or legal requirements)
  • We do not use Google API data for advertising
  • Humans read Google API data only with user consent, for security investigations, legal obligations, or when aggregated/anonymized

6. Data Deletion & Access Revocation

  • In-App Logout (VAU): Deletes stored tokens
  • Google Account: Revoke access at Google security settings
  • Uninstall: Removes all locally stored data

7. Security

  • OAuth 2.0 PKCE authentication
  • Windows DPAPI (safeStorage) token encryption
  • HTTPS-only communication
  • Content Security Policy (CSP)

8. Cookies

We do not currently use cookies. If we begin using them, we will update this policy.

9. Children's Privacy

Our Products are not intended for children under 13.

10. Policy Changes

This policy may be updated as needed. Significant changes will be communicated through the app or website.
